# 价值投资圈博客内容规划

## 四大支柱内容体系

### 1. 大师智慧与实战结合 (Philosophy & Practice)
**频率**：每两周一次
**内容**：挑选一位投资大师的名言或思维模型，结合当前市场现象或个人感悟深度拆解
**示例主题**：
- 芒格的"跨学科思维"在AI投资中的应用
- 巴菲特的"护城河理论"看某行业
- 段永平的"本分"理念与企业管理
- 李录的"知识变现"与投资研究
- 达利欧的"债务周期"与当前经济

### 2. 企业商业模式与护城河分析 (Business Moat)
**频率**：每周一次
**内容**：分析一家公司的商业模式、竞争优势、企业文化（不给出具体买卖建议）
**分析框架**：
- 公司如何赚钱？
- 护城河是什么？有多深？
- 商业模式是否可持续？
- 企业文化和管理层质量
- 财务指标分析

### 3. 读书笔记与"知识变现" (Reading Notes)
**频率**：每周一次
**内容**：分享经典投资书籍章节感悟、优秀财报或致股东信解读
**推荐书籍**：
- 《穷查理宝典》
- 《证券分析》
- 《巴菲特致股东的信》
- 《周期》
- 《投资最重要的事》
- 《文明、现代化、价值投资与中国》

### 4. 初学者投资心理按摩与指南 (Education & Mindset)
**频率**：每月1-2次（市场极端情绪时增加）
**内容**：连载系列，帮助新手建立正确投资心态
**主题方向**：
- 如何保持"平常心"
- 克服情绪化决策
- 投资常识普及
- 风险管理教育

## 内容创作流程

### 1. 选题规划（每月1日）
- 根据四周轮动确定当月主题
- 准备相关书籍、资料、案例
- 制定详细内容大纲

### 2. 内容创作（每周固定时间）
- 周一：资料收集和初步构思
- 周二-周三：深度研究和写作
- 周四：编辑和优化
- 周五：发布准备

### 3. 发布流程
- 使用Hexo生成静态文件
- 提交到GitHub仓库
- 检查GitHub Actions部署状态

## 内容模板

### 大师智慧文章模板
```
标题：[大师姓名]的[理论名称]：[应用场景]

引言：介绍大师名言或理论
理论解析：深入解释该理论
实战应用：结合当前市场或商业现象
个人感悟：自己的理解和体会
总结：核心要点和行动建议
```

### 企业分析文章模板
```
标题：拆解[公司名]的商业模式：它的护城河有多深？

公司概况：基本信息和业务介绍
赚钱逻辑：如何产生收入和利润
护城河分析：竞争优势来源
可持续性：长期发展潜力
风险因素：潜在挑战和风险
总结：商业模式评价
```

## 实施建议

1. **循序渐进**：先从每周一篇开始，逐步增加频率
2. **质量保证**：宁缺毋滥，确保每篇文章都有价值
3. **持续学习**：不断阅读和思考，积累素材
4. **互动交流**：鼓励读者留言讨论，形成社区氛围

这个规划将帮助您建立一个可持续、高质量的内容体系，让博客成为您投资体系的沉淀池和能力圈的拓展记录。
